Date Night

a MEAN stack app built as part of a team that involved using GitHub collaboratively. Date Night consumes several APIs to allow users to plan their date night by selecting a cinema equidistant from 2 locations, see the show times for that cinema and select a restaurant close by and recieved the details of the date by email. The work was divided equally amongst the team so that everyone had experience of working with the whole stack but I was principally for the film times API requests in both the front and back end. The app allows user to upload an image of their date which is uploaded to AWS S3, authentication uses JWT and nodemailer was implimented to automatically send emails.

    Built Using;

  • Node.js
  • MongoDB
  • AngularJS
  • AWS S3
  • Nodemailer

Aurora-Explorer

a full stack app built with a separately deployed back-end built in Rails and front-end built in AngularJS. The app allows the user to see the locations on Earth with the highest probability of seeing the Aurora on a map and to see flights to that location. The details of the trips are saved in a Postgresql database. The app consumes several APIs including Auroras Live and Skyscanner. Authentication using JWT

    Built Using;

  • Rails
  • Angular JS
  • Postgresql
